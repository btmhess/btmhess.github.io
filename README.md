# Project Portfolio

This repository contains the source files for my personal project portfolio,
hosted via GitHub Pages.

The site documents personal engineering and fabrication projects with:
- goals and requirements
- build results and specifications
- materials / parts lists
- image galleries

## Structure

- `_projects/` — individual project folders
- `index.html` — main project list
- `_layouts/` — site templates
- `assets/` — CSS and JavaScript
